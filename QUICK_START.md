# vipPolice 快速入门指南

## 📱 如何在 iPhone 上运行这个 App

### 方法一：使用 Xcode（推荐）

#### 步骤 1: 安装 Xcode
1. 在 Mac 上打开 App Store
2. 搜索并下载 **Xcode**（免费，需要 macOS）
3. 安装完成后打开 Xcode

#### 步骤 2: 创建项目
1. 打开 Xcode
2. 选择 **File → New → Project**
3. 选择 **iOS → App**
4. 填写项目信息：
   - **Product Name**: `vipPolice`
   - **Team**: 选择你的 Apple ID（如果没有，点击 Add Account 添加）
   - **Organization Identifier**: `com.yourname`（随意填写）
   - **Interface**: 选择 **SwiftUI**
   - **Language**: 选择 **Swift**
   - **Storage**: 不勾选 Core Data
5. 点击 **Next**，选择保存位置

#### 步骤 3: 导入代码文件
1. 在 Xcode 左侧项目导航器中，删除自动生成的 `ContentView.swift`
2. 右键点击项目名称 → **New Group**，创建以下文件夹：
   - `Models`
   - `Views`
   - `Services`
3. 将我创建的文件复制到对应文件夹：
   ```
   Models/ → Benefit.swift, MemberAccount.swift
   Views/ → MonthlyFocusView.swift, AllAccountsView.swift, 等
   Services/ → OCRParser.swift, DataManager.swift, NotificationManager.swift
   根目录 → ContentView.swift, vipPoliceApp.swift
   ```
4. 替换 `Info.plist` 文件

#### 步骤 4: 配置和运行
1. 在 Xcode 顶部，选择目标设备：
   - **真机**：连接你的 iPhone，选择你的设备
   - **模拟器**：选择 iPhone 15 Pro 或其他模拟器
2. 点击 **▶️ 运行按钮**（或按 ⌘R）
3. 首次运行可能需要：
   - 在 iPhone 上信任开发者证书（设置 → 通用 → VPN与设备管理）
   - 允许相册和通知权限

## 🎯 App 使用教程

### 第一次使用

#### 1. 添加第一个会员账号

**场景**：你想管理美团会员的红包和优惠券

1. **截取会员页面**
   - 打开美团 App
   - 进入"我的" → "会员中心"
   - 截图保存（确保包含平台名称、红包金额、到期时间）

2. **导入到 vipPolice**
   - 打开 vipPolice App
   - 点击右上角 **+** 按钮
   - 选择刚才的截图
   - 等待识别（约 2-3 秒）

3. **确认识别结果**
   - App 会显示识别到的信息
   - **重要**：仔细核对每一项：
     - ✅ 平台是否正确（美团）
     - ✅ 金额是否准确（¥5）
     - ✅ 到期时间是否正确（2026-03-15）
   - 如有错误，点击该权益进行编辑
   - 确认无误后，点击右上角"确认"

4. **完成！**
   - 权益已保存
   - 如果即将过期，会出现在"本月待办"

### 日常使用场景

#### 场景 1: 查看本月需要处理的任务

**目的**：不错过任何优惠

1. 打开 App，默认显示"本月待办"页面
2. 查看两个部分：
   - **待领取**：每月需要领取的红包、券（如京东每月 1 日的 PLUS 券）
   - **待使用**：本月即将过期的优惠券
3. 完成后，点击前面的 ⭕ 标记为已完成
4. 已完成的项目会变灰，不再提醒

#### 场景 2: 查看所有会员权益

**目的**：管理多个平台的会员

1. 点击底部"全部权益" Tab
2. 看到所有已添加的平台（淘宝、京东、美团等）
3. 点击任意平台，查看详细权益列表
4. 左滑可以删除不需要的账号

#### 场景 3: 开启到期提醒

**目的**：自动收到通知，不会忘记

1. 点击底部"设置" Tab
2. 打开"到期提醒"开关
3. 允许通知权限
4. 完成！App 会在以下时间提醒你：
   - 周期性权益：重置日前一天上午 9:00
   - 一次性权益：过期前 5 天

## 💡 使用技巧

### 截图建议

**好的截图** ✅
- 清晰可读
- 包含平台名称（美团、京东等）
- 包含金额（¥5、¥10）
- 包含日期（2026-03-15）
- 光线充足

**不好的截图** ❌
- 模糊不清
- 没有平台名称
- 没有到期时间
- 过暗或过亮

### 手动校验的重要性

**为什么要手动确认？**
- OCR 识别率约 80-90%
- 可能识别错误：
  - 金额：¥5 识别成 ¥8
  - 日期：2026-03-15 识别成 2026-03-18
  - 限制：满 30 可能漏掉

**如何快速校验？**
1. 重点检查：金额、日期
2. 对比原截图
3. 有疑问就点击编辑

### 定期维护

**每周一次**（建议）
1. 打开"本月待办"
2. 标记已使用的权益
3. 删除已过期的账号

**每月初**
1. 检查周期性权益是否已领取
2. 更新新的会员信息

## ❓ 常见问题

### Q1: 识别失败怎么办？
**A**: 
1. 确保截图包含平台名称（美团、京东等）
2. 确保截图清晰
3. 重新截图再试
4. 如果还是失败，可能该平台暂不支持

### Q2: 识别结果不准确？
**A**: 
1. 在确认页面点击权益
2. 手动编辑金额、日期
3. 保存即可

### Q3: 没有收到提醒？
**A**: 
1. 检查"设置"中是否开启"到期提醒"
2. 检查 iPhone 设置 → 通知 → vipPolice 是否允许
3. 确保权益未标记为"已使用"

### Q4: 如何修改已保存的权益？
**A**: 
1. 进入"全部权益"
2. 点击对应平台
3. 点击权益前的圆圈或查看详情
4. 目前不支持直接编辑，建议删除后重新添加

### Q5: 支持哪些平台？
**A**: 
目前预设支持：
- ✅ 淘宝/天猫
- ✅ 京东
- ✅ 美团
- ✅ 饿了么

其他平台可以尝试，但可能需要更多手动调整。

### Q6: 数据会上传到云端吗？
**A**: 
不会！所有数据仅存储在你的 iPhone 上，完全本地化，保护隐私。
